## 🛠️ 一键部署与升级（直接复制运行）

### 方案 A：通用裸机/虚拟机一键部署 (支持银河麒麟/CentOS/RHEL/Ubuntu)

```bash
#!/usr/bin/env bash
# ==============================================================================
# TDSQL-SQLCheck V1.2.0.1 一键部署与升级自动化脚本
# ==============================================================================
set -e

APP_DIR="/opt/tdsql-sqlcheck"
RELEASE_TAR="/tmp/tdsql-sqlcheck-v1.2.0.1-source.tar.gz"

echo "==== [Step 1/5] 创建并清理目录 ===="
sudo mkdir -p ${APP_DIR}
sudo systemctl stop tdsql-sqlcheck || pkill -f "uvicorn backend.main:app" || true

echo "==== [Step 2/5] 部署代码解压 ===="
if [ -f "${RELEASE_TAR}" ]; then
    sudo tar -xzf ${RELEASE_TAR} -C ${APP_DIR} --strip-components=1
else
    echo "[Info] 未发现离线包，保持当前源码目录运行"
fi

cd ${APP_DIR}

echo "==== [Step 3/5] 离线Python虚拟环境准备 ===="
if [ ! -d "venv" ]; then
    python3 -m venv venv
fi
source venv/bin/activate

# 优先从本地/wheels依赖包安装
if [ -d "../wheels" ]; then
    pip install --no-index --find-links=../wheels -r requirements.txt
else
    pip install -r requirements.txt
fi

echo "==== [Step 4/5] 启动 Backend FastAPI 服务 ===="
nohup python -m uvicorn backend.main:app --host 0.0.0.0 --port 8000 > backend_restart.log 2>&1 &

echo "==== [Step 5/5] 健康检查验货 ===="
sleep 3
HEALTH_CHECK=$(curl -s http://127.0.0.1:8000/health || echo "FAIL")
echo "健康探针返回: ${HEALTH_CHECK}"

if [[ "${HEALTH_CHECK}" == *"1.2.0.1"* ]]; stream
    echo "🎉 [Success] TDSQL SQL审核工具 V1.2.0.1 部署成功并成功拉起！"
else
    echo "❌ [Error] 启动失败，请检查 backend_restart.log"
    cat backend_restart.log | tail -n 20
    exit 1
fi
```

### 方案 B：Docker / Docker Compose 离线容器一键部署

```yaml
# docker-compose.yml
version: '3.8'

services:
  tdsql-sqlcheck:
    image: tdsql-sqlcheck:v1.2.0.1
    container_name: tdsql-sqlcheck-app
    restart: always
    ports:
      - "8000:8000"
    environment:
      - AUTH_ENABLED=true
      - ADMIN_INITIAL_PASSWORD=Admin@1234
      - DB_TYPE=sqlite
    volumes:
      - ./data:/app/data
      - ./logs:/app/logs
    healthcheck:
      test: ["CMD", "curl", "-f", "http://localhost:8000/health"]
      interval: 10s
      timeout: 5s
      retries: 3
```

**一键启动命令**：
```bash
docker-compose up -d
docker-compose ps
```
